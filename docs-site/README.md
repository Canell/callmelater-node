# CallMeLater Documentation

This repository contains the **official documentation** for the CallMeLater API and platform.

📘 **Read the docs:** https://docs.callmelater.io  
🌐 **Product website:** https://callmelater.io

CallMeLater is a developer-focused service for scheduling future actions such as delayed HTTP calls and human-interactive reminders, with reliability, auditability, and clarity as first principles.

---

## Purpose of this repository

This repository serves as:

- The **source of truth** for CallMeLater’s public documentation
- A **transparent reference** for API behavior, concepts, and guarantees
- A **versioned history** of documentation changes

It does **not** contain application source code.

## About CallMeLater

CallMeLater helps developers answer the question:

> “Do this later — and don’t lose it.”

Instead of fragile cron jobs or custom scheduling logic, CallMeLater provides a reliable API to:
- schedule HTTP calls
- send reminders that require human confirmation
- track execution, retries, and outcomes

Learn more at https://callmelater.io
